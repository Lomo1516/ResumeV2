Project 2 Details

Store Checkout & Return Queue Simulation (Java)

Project Overview

This project is a Java simulation of store checkout and return queues. It models customer arrival, queue processing, and return handling, providing a dynamic analysis of store operations and efficiency.

Skills & Technologies Used

  • Java Programming
  • Object-Oriented Design
  • Queues & Data Structures
  • Simulation & Console Output

Challenges & Learning

Through this project, I learned how to simulate real-world processes, handle multiple customer scenarios, and optimize queue management. Testing and debugging ensured realistic behavior in various store conditions.
