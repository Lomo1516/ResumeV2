Sales Transaction Statistics Program

Project Overview

Developed a C program that generates and analyzes sales transaction data. The program processes large sets of numerical data and calculates statistical values such as minimum, maximum, total, and average. This project demonstrates fundamental programming concepts, arrays, functions, modular programming, and basic statistics. I also led a team of three, coordinating efforts and implementing core calculation logic.
